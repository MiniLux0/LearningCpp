# Exercises — LearningCpp

Ejercicios de practica basados en las lecciones L06-L15.

## Como usar

```bash
# Compilar todos los ejercicios
make

# Compilar y ejecutar uno especifico
make run-E01_VariableTypes

# Ejecutar con AddressSanitizer
make run-asan-E01_VariableTypes

# Limpiar builds
make clean
```
