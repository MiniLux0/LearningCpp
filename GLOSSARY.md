# C++ Glossary

C++ terms I am learning, explained in simple language.

---

## A

- **Scope** — Where a variable "exists" and can be used. If declared inside a function, it only lives there.

## B

- **Bit** — The smallest unit of data in a computer. It can only be 0 or 1.
- **Byte** — 8 bits grouped together. It is the basic unit of memory.

## C

- **Compiler** — Program that translates our C++ code into machine language (binary) so the computer can understand it.
- **Concatenation** — Joining two or more strings with the `+` operator.
- **Constructor** — Special function that runs automatically when an object is created. Used to initialize data.

## D

- **Declaration** — Telling the compiler that a variable exists, with its type and name.
- **Overflow** — When a number exceeds the maximum range its type can hold.

## E

- **Enum** — Data type that allows defining a set of named constants.
- **Expression** — Any combination of values and operators that produces a result.

## F

- **Function** — Reusable block of code that performs a specific task.
- **Framework** — Set of tools and libraries that simplify software development.

## H

- **Header** — `.h` or `.hpp` file containing declarations of functions, classes, or variables that other files can use.

## I

- **IDE (Integrated Development Environment)** — Program that combines editor, compiler, and debugger in one (e.g., VSCode, Code::Blocks).
- **Initialization** — Giving a variable its initial value when declaring it.
- **Instance** — Object created from a class.

## L

- **CLI (Command Line Interface)** — Text interface to interact with the computer (e.g., PowerShell, Terminal).

## M

- **Memory** — Space in the computer where data is stored while the program runs.

## O

- **Operator** — Symbol that performs an operation (e.g., `+`, `-`, `==`, `&&`).
- **Object** — Instance of a class. Contains data and behavior.

## P

- **Pointer** — Variable that stores the memory address of another variable.
- **Template** — Generic function or class that works with different data types.

## R

- **Reference** — Alias or alternative name for an existing variable.
- **Return** — Value that a function gives back when it finishes.

## S

- **String** — Text chain (collection of characters).
- **Overloading** — Creating multiple functions with the same name but different parameters.

## T

- **Data Type** — Defines what kind of value a variable can hold (int, float, char, bool, etc.).

## V

- **Variable** — Name that stores a value in memory.
