# Getting Started with C++

These are my personal study notes from **John Purcell's "C++ Tutorial for Complete Beginners"** on Udemy. This first section covers the absolute basics — what C++ is, how computers handle numbers, and how to set up the development environment.

This repository is designed to serve two purposes:
- A **study guide** for anyone learning C++ from scratch, with organized notes and code examples by topic
- A record of my **personal progress** so others can see how I am advancing through the course

---

## L01 — Introducing C++

- C++ is a general-purpose programming language created by **Bjarne Stroustrup** in the late 1970s
- It was built on top of the C language, adding features like classes and object-oriented programming
- C++ is used in game development, operating systems, embedded systems, and high-performance applications
- It is popular because it gives the programmer a lot of control over hardware and memory
- Many modern languages (like Java, C#, and Python) were influenced by C++

---

## L02 — Screen Resolution

- A computer screen is made up of tiny dots called **pixels**
- Screen resolution (e.g., 1920×1080) tells us how many pixels are in the width and height
- This is a practical example of how computers use numbers to represent real-world things
- Understanding how numbers work in computing is fundamental to programming
- Everything on a computer — images, text, sound — is stored as numbers

---

## L03 — Setup and Installation

- To write C++ code, you need a **compiler** (translates human-readable code into machine language)
- An **IDE** (Integrated Development Environment) makes writing, compiling, and running code easier
- Popular free options include **Code::Blocks**, **Visual Studio**, and **CLion**
- I am using **Visual Studio Code** with the C/C++ extension — it is lightweight and works great
- John Purcell recommends Code::Blocks for beginners because it is simple and lightweight
- Once installed, you can write your first `.cpp` file and compile it to see output
