## ⚙️ Entorno

- **Compilador:** GCC 15.2.0 (WinLibs) en Windows / g++ en Linux
- **Editor:** VSCode
- **Estándar:** C++17
- **Flags:** `-std=c++17 -Wall -Wextra -Wshadow -Wpedantic -g`

### Compilar un archivo
```bash
g++ -std=c++17 -Wall archivo.cpp -o programa
./programa
```
