## Environment

- **Compiler:** GCC 15.2.0 (WinLibs) on Windows / g++ on Linux
- **Editor:** VSCode
- **Standard:** C++17
- **Flags:** `-std=c++17 -Wall -Wextra -Wshadow -Wpedantic -g`

### Compile a file
```bash
g++ -std=c++17 -Wall filename.cpp -o program
./program
```
