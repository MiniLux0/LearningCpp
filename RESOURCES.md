# Useful Resources

Links and tools that help me learn C++.

---

## Documentation

- [cppreference.com](https://cppreference.com/) — Official C++ reference. Complete and technical.
- [C++ Tutorial (cplusplus.com)](https://www.cplusplus.com/doc/tutorial/) — Tutorials organized by topic.
- [LearnCpp.com](https://www.learncpp.com/) — Free and comprehensive tutorial, step by step.

## Main Course

- [John Purcell — C++ Tutorial for Complete Beginners (Udemy)](https://www.udemy.com/course/free-ncurses-programming-tutorial-for-beginners/) — The course I am following. Clear explanations and practical exercises.

## Tools

- [VSCode](https://code.visualstudio.com/) — The editor I am using. Lightweight and customizable.
- [C/C++ Extension for VSCode](https://marketplace.visualstudio.com/items?itemName=ms-vscode.cpptools) — Extension for autocompletion, debugging, and compilation.
- [GCC via WinLibs](https://winlibs.com/) — GCC compiler for Windows, easy to install.
- [Godbolt Compiler Explorer](https://godbolt.org/) — See the assembly your code generates online.

## Practice

- [LeetCode](https://leetcode.com/) — Programming exercises to practice logic.
- [HackerRank](https://www.hackerrank.com/domains/cpp) — C++ specific challenges.
- [Exercism (C++ Track)](https://exercism.org/tracks/cpp) — Guided exercises with mentoring.

## Videos and Channels

- [The Cherno (YouTube)](https://www.youtube.com/c/TheChernoProject) — Explains C++ and game development very well.
- [C++ Weekly (YouTube)](https://www.youtube.com/c/cppweekly) — News and tips on modern C++.
- [Jason Turner (YouTube)](https://www.youtube.com/c/jasonturner) — Practical C++ and best practices.

## Books *(future references)*

- *A Tour of C++* — Bjarne Stroustrup (the creator of C++)
- *Effective Modern C++* — Scott Meyers
- *C++ Primer* — Stanley Lippman
